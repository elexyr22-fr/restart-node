# Comment installer nouvelle Node 

1) Connecte toi au SSH:

User: `root`

2) Ecris la commande:

```
nano /etc/pterodactyl/config.yml
```

3) Remplace le port **443** par:
```
8080
```

5) Ensuite tape:

```
reboot -f
```   
5) Pour fermer avec nano, faite **CTRL + X**, puis **Y**, puis **ENTRER**

La node est restart gg all !
